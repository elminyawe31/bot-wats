# ELMINYAWE Bot v4.0.0

WhatsApp Group Management Bot by **Yaseen ELMINYAWE**

## Features
- 100+ commands for group management
- Arabic/English bilingual support  
- Sub-bot (Jadibot) system
- AI commands (GPT, image generation)
- Media downloader (YouTube, Instagram, TikTok, Facebook, Spotify)
- Group protection (antilink, antibadword, anticall, antidelete)
- Fun commands (games, memes, stickers)
- And much more!

## Installation

1. Clone the repo
2. Run `npm install`
3. Set your number in `settings.js`
4. Run `node index.js`
5. Enter the pairing code on WhatsApp
